- **Cinematic UI/UX**: A refined design system with warm porcelain tones, deep slate accents, and diffused shadows.

## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Shantanux0/LUMIERE.git
   ```

2. Navigate to the project directory:
   ```bash
   cd LUMIERE/FurFrontend
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
